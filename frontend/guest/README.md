# CrisisSync - Guest Frontend

## Part A - Built by [Purbasa Pramanik]

This is the Guest Frontend for the CrisisSync Hotel Emergency Response System.

## Pages
- **Room Select** - Guest enters their room number
- **SOS Page** - Guest selects emergency type (Fire, Medical, Threat)
- **Confirmation** - Shows "Help is on the way!" after SOS is sent

## How to Run
1. Navigate to frontend/guest
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Backend Connection
Connects to backend API at `http://127.0.0.1:8000/emergency/report`
